# Stamp Exchange Registry

This file serves as a global registry for the Philatelist Community. Collectors can list stamps they are looking for (ISO) or have available for trade (AFT).

## How to List
Add your entry in the following format:
`- [TYPE] [COUNTRY] [YEAR] [NAME] - [CONDITION] - [CONTACT/LOCATION]`
